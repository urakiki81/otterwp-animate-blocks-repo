Otterwp Animate Gutenberg blocks plugin for WordPress.

== Description ==
Animate Gutenberg blocks when they scroll into view.

* Works with all existing Gutenberg blocks
* Choose from 30+ different animations
* Configure animation with custom delays and durations

= Requirements =
* WordPress >= 5.0
* PHP >= 5.6

= Further Information =

* Documentation: [https://www.otterwp.com/support]
* Changelog: [https://www.otterwp.com/support]

== Installation ==

1. Upload the `otterwp-animate-blocks` directory into the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. Add the license key you get from freemius.
4. Toggle On `Otterwp Animate Blocks` on any core block in Gutenberg editor.
5. Add `Otterwp Animate Blocks` block around other block in Gutenberg editor.

== Frequently Asked Questions ==

= How does it work? =
Otterwp Animate blocks extends all gutenberg blocks to enable and add animation. Otterwp Animate blocks also inclues a It wrapper own blocks base on the `InnerBlocks` feature of Gutenberg. You can add as many blocks inside this wrapper block as you want.
